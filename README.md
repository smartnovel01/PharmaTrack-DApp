# PharmaTrack DApp

## Description
PharmaTrack is a blockchain-based pharmaceutical supply chain tracking system.

## Features
- Add medicine
- Transfer ownership
- View medicine details

## Technologies
- Solidity
- Remix IDE
- MetaMask
- Ethers.js
- HTML, CSS, JavaScript
- Ethereum Sepolia

## How to Run
1. Deploy smart contract using Remix
2. Connect MetaMask (Sepolia network)
3. Open index.html in browser
4. Interact with the DApp
